Google OAuth App - Consent Screen Support Files

These files include:

1. index.html - Homepage of the app
2. privacy.html - Privacy policy as required by Google
3. terms.html - Terms of service (optional but recommended)
4. README.txt - This description file

Instructions:
- Upload these files to GitHub or Netlify.
- Use the links in your OAuth consent screen setup.
- Make sure the domain (e.g., github.io) is added to the Authorized Domains in Google Cloud Console.
